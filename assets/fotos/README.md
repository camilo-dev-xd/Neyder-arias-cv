# 📷 Carpeta de Fotos

Aquí puedes agregar todas tus fotografías personales, fotos de perfil, fotos de equipos de trabajo, o cualquier foto que desees usar en tu portafolio.

## Estructura Recomendada

```
fotos/
├── perfil/
│   └── tu-foto-perfil.jpg       # Foto de perfil profesional
├── proyectos/
│   ├── gastro-coffee/
│   ├── badminteam/
│   └── factoria/
└── equipo/
    └── equipo-trabajo.jpg
```

## Cómo Usar

1. Coloca tus fotos aquí
2. Para referenciarlas en el HTML, usa: `assets/fotos/nombre-foto.jpg`

Ejemplo:
```html
<img src="assets/fotos/perfil/mi-foto.jpg" alt="Mi foto de perfil">
```

## Formato Recomendado

- **Formato**: JPG (para fotos) o PNG (para imágenes con transparencia)
- **Tamaño máximo**: 300KB por foto (usa TinyPNG.com)
- **Dimensiones**: 
  - Foto de perfil: 300x300px
  - Fotos de proyectos: 1200x800px o más
